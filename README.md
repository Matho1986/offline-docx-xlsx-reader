# Offline DOCX/XLSX Reader

## Funktionen

### DOCX

- Anzeige
- Volltextsuche
- Kopieren & Teilen
- Drucken (Android Print Framework)
- Zoom (2-Finger-Geste)

### XLSX

- Anzeige + Sheet-Auswahl
- Volltextsuche
- Kopieren & Teilen
- Drucken
- Zoom (2-Finger-Geste)

### PDF (Bonus)

- Anzeige
- Zoom (2-Finger-Geste)
- Drucken (wenn implementiert)
- Hinweis: PDF wird als Seite/Bild gerendert (keine Suche, kein Text kopieren)
