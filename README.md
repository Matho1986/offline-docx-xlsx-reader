# Offline DOCX/XLSX Reader

## Eigenschaften

- Komplett offline
- Werbefrei
- Keine Cloud / keine Tracker
- Keine fremden Rendering-Services (kein Google Docs / Office Online)
- Datenschutzfreundlich
- Rotation-stabil

## Funktionen

### DOCX

- Anzeige
- Volltextsuche
- Kopieren & Teilen
- Drucken (Android Print Framework)
- Zoom (2-Finger-Geste)

### XLSX

- Anzeige + Sheet-Auswahl
- Volltextsuche
- Kopieren & Teilen
- Drucken
- Zoom (2-Finger-Geste)

### Textformate

Unterstützte Formate:

- `.txt`
- `.json`
- `.log`
- `.csv`
- `.md` (aktuell als Plain-Text)

Für diese Formate:

- Anzeige im integrierten Text-Viewer
- Suche
- Alles kopieren & Teilen
- Bearbeitung und **Speichern unter…**

### Neues Dokument erstellen

Über das Overflow-Menü → **Neu**:

- Neues TXT
- Neues JSON
- Neues LOG
- Neues CSV
- Neues MD

## Sprache
