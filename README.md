# Offline DOCX/XLSX Reader

## Eigenschaften

- Komplett offline
- Werbefrei
- Keine Cloud / keine Tracker
- Keine fremden Rendering-Services (kein Google Docs / Office Online)
- Datenschutzfreundlich
- Rotation-stabil

## Funktionen

### DOCX

- Anzeige
- Volltextsuche
- Kopieren & Teilen
- Drucken (Android Print Framework)
- Zoom (2-Finger-Geste)

### XLSX

- Anzeige + Sheet-Auswahl
- Volltextsuche
- Kopieren & Teilen
- Drucken
- Zoom (2-Finger-Geste)

### PDF (Viewer)

- Anzeige
- Zoom (2-Finger-Geste)
- Drucken (wenn implementiert)
- Hinweis: Fokus liegt auf stabiler Darstellung, nicht auf Textanalyse
  (keine Volltextsuche oder Kopierfunktion)
